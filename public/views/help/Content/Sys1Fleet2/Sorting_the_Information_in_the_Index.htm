Sorting Information in the Grid View Table

To the right of each column row heading is an arrow that allows you to re-sort the default order of the column. The sort order of the Total Alarms column reflects the settings on the User Preference page. If you re-sort the column, the Total Alarms column displays alarms from the highest to the lowest.

  NOTE: If you change the sort order of one column, all columns are re-sorted.

To re-sort the default order of the information in a column:

  1. Click the column title. A white arrow appears to the right of the column title.
  2. Click the arrow . The column contents are sorted in reverse order (see the following figure).
