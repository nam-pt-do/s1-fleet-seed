Introducing the Dashboard Module

The Dashboard module is displayed by default after you log in to System 1 Fleet Management. The Dashboard provides icons and controls that you can use to navigate the System 1 Fleet Management portal. The main feature of the Dashboard page is the Overview table. The table displays health data about the conditions of the Enterprises, Trains, Machines and Assets for the selected customer's Fleet. The following figure shows an example of a typical Dashboard module, in which a customer has been chosen and a folder is selected from the hierarchy list.

Dashboard Module Icons and Controls

The information in the following table describes the icons and controls that apply to the Dashboard page in general.

Icon or Control Name Description
Customer List This list shows the names of customers whose systems you can access.
Hierarchy List This list shows the Fleet hierarchy for the selected customer. The default view is at the Fleet Asset level. SmartSignal applications that are unmapped are included at the end of the list.
Overview tab Click the Overview tab to view health data about the selected customer's Enterprises, Trains, or Machines.
Alarms tab Click the Alarms tab to launch a new page that contains an Alarms List table and other information about the health of the selected customer's Enterprises, Trains, or Machines
Analysis tab Click the Analysis tab to launch a new page on which you can generate a chart that plots points and variables.
Refresh button Click the button to refresh the health data displayed on the Overview or Alarms page.
User Profile list Click the down arrow to the right of your User Profile name to select and view Account Profile settings, view and modify Preferences, and Sign Out of System 1 Fleet Management. You have the option to change the password in your Account Profile.

Overview Table Icons and Controls

The information described in the following table apply to the Overview Table that is displayed by default on the Dashboard module.

Item Name Description
Search box Enter search terms and click the magnifying glass to locate information in the Overview table.
Assets View button Click to display an overview of health data for all of the assets in the selected Enterprise. Click the button again to restore the previous view of the Overview table.
Gauges icon Click to open a new page that displays an overview of Enterprise health in an array of gauges. Click the Index icon to restore the default view
Index icon Click to open a new page that displays an overview of Enterprise health in a table format.
