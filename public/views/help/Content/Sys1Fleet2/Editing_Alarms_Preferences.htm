Editing Alarms Page Preferences

The topics in this section describe how to configure preference settings for the Alarms page (see following figure).

Default Chart Type After Navigating to the Alarm Page

Click one of the following chart types:

  • Bar Chart
  • Line Chart
  • Alarm Plot

Alarms Grid

The settings in this section control the appearance and organization of the data displayed after you click the alarms grid on the Alarms page.

Default Number of Entries

Click one of the following parameters to set the default number of the rows displayed on each page in the Alarms table.

  • 10
  • 20
  • 30
  • 50
  • 100

Default Sort

Click one of the parameter names below to control the default sort order of the how the information displayed in the Alarms List.

  • Last Occurrence
  • First Occurrence
  • Data Source
  • Fleet Level
  • Source Level
  • Asset Path
  • Point/Variable
  • Type/Message
  • Count
  • Density
  • Source ID
  • Event ID
  • Acknowledged
  • Time of Acknowledgement
  • Active
  • Train/Machine
  • Event Category

Default Sort Order

Click one of the following parameters:

  • Ascending
  • Descending
