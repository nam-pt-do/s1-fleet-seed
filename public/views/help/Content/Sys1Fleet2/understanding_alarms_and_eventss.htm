Alarms and Events

Alarms tab

The Alarms tab has two sections. The top section contains line charts, bar charts, or alarm plots. The lower section contains an Alarms List table. Use the Alarms tab to;

  • Generate a line or bar chart showing Subgroup or Event Type alarm count and alarm severity for a selected Fleet, Enterprise, Folder, Train, or Machine.
  • Display an alarm plot(s): Trend or Spectrum.
  • View an Alarms List.
  • Export the Alarms List table to a .csv file.
  • Select the columns displayed in the Alarms List table.
  • Select filter parameters to create charts that show specific, customized alarm information.
  • Save the bar, line, or alarm plot as a .PNG file, or print the plot(s) from the Alarms page.
