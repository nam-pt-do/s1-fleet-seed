Creating a User

Follow the steps in this procedure to create a System 1 Fleet Management user.

  NOTE: This procedure is only required if the customer is not using LDAP or Active Directory authentication.
  1. From the home page, click Administration. The Users page is displayed.
  2. From the Users page, click Add New User. The Add New User page is displayed. You must enter data in every field on the page.

  3. In the Username box, enter a name for the new user. The Username must be between eight and 20 characters. The new user will enter this information in the Username field when logging in to System 1 Fleet Management.

  4. Select a Role. The roles are described below.

    User - People who are assigned to a User role cannot view or use the Administration module. They can access the customers assigned to them when their user account was created.

    Administrator- People who are assigned to an Administrator role can see and access only the customers assigned to them when their user account was created. They can access the Administration module. Administrators can create, edit and delete Users, and administer Enterprises.

    SuperAdministrator- People who are assigned to this role can see and access all customers and tenants in a multi-tenant configuration. They can also access the Administration module and perform all tasks associated with the Administrator role.

  5. Assign a Name to the new user. The name you assign is displayed in the Account Profile box. The user can change the Name by modifying their Account Profiles.
  1. Enter an Email address for the new user. The user can change the Email address by modifying their Account Profile.
  2. Click OK. The Users page is displayed. Information about the new user is included the table on the Users page.
