Getting Started

Getting Started

This section explains how to start using System 1 Fleet Management and how to logout after a session.

  NOTE: If the system is configured to use LDAP or AD, you do not need to have a separate System 1 Fleet Management username and password.

Signing in to System 1 Fleet Management

You must have Internet access, the System 1 Fleet Management URL and a Username and Password configured by the system administrator before you can access the product.

To sign in to System 1 Fleet Management, complete the following steps;

  1. Launch a Web browser.
  2. Enter the System 1 Fleet Management URL. The sign-on page is displayed.
  3. Enter your Username and Password.
  4. Click Sign in. The Dashboard module is displayed, and opens to the Overview tab.

Performing Tasks

The types of tasks that you can perform depend on the role assigned when the system administrator configured your account. For example, if your role is assigned as a User, you cannot perform any administrative tasks and the Administration module on the Dashboard page will not be visible to you.
