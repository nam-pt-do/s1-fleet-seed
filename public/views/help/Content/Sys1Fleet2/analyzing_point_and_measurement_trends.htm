Viewing Analysis Statistics

Generating an Analysis Plot

An analysis plot displays a plot generated by selecting variables for a specific time and date interval for a selected machine or train. The Analysis Plot feature is only available for static variables in System 1. After you generate an Analysis plot, you can save and retrieve plot session endpoint and variable settings in the database. You can print an analysis plot, or save the plot as a .PNG graphic file.

Plot session information consists of: 

  • Variables
  • Points
  • Start Date
  • End Date
  • Start Time
  • End Time

To generate an Analysis Plot, complete the following steps;

  1. From the Dashboard page, click the Analysis tab. The Analysis page is displayed.
  2. Select a Customer.
  3. Select a System 1 asset (train or machine) from the Hierarchy menu. The Analysis page is refreshed. The left pane displays a list of points associated with the selected asset point. The right pane contains the Variables for the selected Points.
  4. Click a point in the left pane. The Analysis page is refreshed. The right pane displays a list of the variables associated with the selected point.
  5. Click one or more check boxes in the Variables pane. Each selected variable is shown as a separate line in the Analysis plot.
  6. Click the calendar icon and click a date. The Start Date box is automatically populated with the selected date.
  7. Click the calendar icon and click a date. The End Date box is automatically populated with the selected date. For example, if you selected September 14, 2014 as the Start Date and selected September 18 as the End Date, the Analysis plot would plot the selected variables over a four-day period.
  8. In the Start Time box, enter the time that marks the start of the time interval for which you want to plot the variables. The format to enter the time is HHMM. Option: Click the clock icon and select a time. The Start Time box is automatically populated with the selected time.
  9. In the End Time box, enter the time that marks the end of the time interval for which you want to plot the variables. The format to enter the time is HHMM. Option: Click the clock icon and select a time. The End Time box is automatically populated with the selected time.
  10. Click Plot Selected Variables. The lower half of the Analysis page displays the plot. The X axis of the plot displays the selected Start and End Dates and the selected Start and End times. The following figure shows an example of a plot.
