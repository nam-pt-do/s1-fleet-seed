Dashboard

Introducing the Dashboard Module

The Dashboard module is displayed by default after you log in to System 1 Fleet Management. The Dashboard provides icons and controls that you can use to navigate the System 1 Fleet Management portal. The main feature of the Dashboard is the grid view table. The table displays health index data about the sites, folders, trains, and machines in a selected customer's Fleet. The following figure shows an example of a typical Dashboard module that is similar to one you might view after logging in.

Selecting a Fleet and Enterprise

After logging in, if one customer exists, the grid view will show all the enterprises under your Fleet. To select a customer and a Fleet, complete the following procedure;

  1. Click the down arrow in the Fleet List box . The customer names displayed in the list box are defined by the System 1 administrator when the System 1 Fleet Management administrator creates your account.
  2. Click the down arrow in the Select drop-down list . A list of sites associated with the selected customer is displayed.
  3. Select a site. The names of folders, trains and machines within the selected site are displayed.
  4. Click the right-facing arrows to expand the contents of the folders or machines.

Sorting Information in the Grid View Table

To the right of each column row heading is an arrow that allows you to re-sort the default order of the column. The sort order reflects the settings on the User Preference page.

  NOTE: If you change the sort order of one column, all columns are re-sorted.

To re-sort the order of the information in a column, complete the following steps;

  1. Click the column title. A white arrow appears to the right of the column title.
  2. Click the column title again. The column contents are sorted in reverse order (see the following figure).

Understanding the Icons in the Status Column

The status column in the Grid View can display a red, green, or gray bar-shaped icon. The information provided by the icon is different, depending on the selection you made when navigating the hierarchy, or drilling down into the Fleet using the links in the Name column.

Icon Selection Description
Site or folder This green icon indicates that the Fleet Web server is receiving communications, as expected, from their corresponding enterprise server.
Train or machine This green icon indicates that the selected train or machine is running.
