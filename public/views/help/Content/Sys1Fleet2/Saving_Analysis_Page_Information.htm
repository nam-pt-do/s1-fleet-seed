Saving Analysis Plot Information

The Analysis page includes features that you can use to:

  • Save analysis plot settings in the System 1 Fleet Management database
  • Retrieve the settings and generate the plot.
  • Save the plot as a .PNG file.
  • Print the plot from the Analysis page.

 

To do this Follow this step

Save a copy of the Analysis plot settings. The saved settings are the ones used to generate the most-recently generated Analysis point, including::

  • Selected Point(s)
  • Selected Variable(s)
  • Start /End Dates
  • Start/End/Times
 Click the icon. The saved settings are stored in the database.
Regenerate a plot from saved settings Click the icon. The page is updated with the saved settings used to generate the previous plot.
Save the current analysis plot as a .PNG file Click the Chart Context Menu icon and select Save as PNG. After you save the plot, the file is automatically stored in the computer's Downloads folder. The default file name is download(x), where x=the sequence of the downloaded plots. No particular file type is assigned to the plot. To view the saved plot, right-click the plot file name and click Open. Select a graphics viewing application, for example, Windows Photo Viewer, to display the saved plot.
Print the current analysis plot Click the Chart Context Menu icon and select Print. The browser displays a page that contains print parameters. Configure the print parameters for your site and click Print.
