Using the Alarms List Filter Feature

You can filter the data displayed in the Alarm List and in the bar, line and alarm plots by filtering on one or more data points in the Alarm List table columns.

The following figure represents an example of a typical Alarms List table before any filter parameters are applied. The actual table columns and rows you see depend on the selections you make from the Show/Hide column list.

View a bar chart of medium-level System 1 alarms by Subgroup from the last month.

After selecting the categories, icons and filter parameters described in the following two sections, the Alarms table and the bar chart will be updated to reflect the data described in the scenario and shown in the following figure.

 

Preparing to use the Filter Feature

 

To prepare for filter feature use, complete the following steps;

  1. Click Alarms from the Dashboard section of the page.
  2. Click the bar chart icon.
  3. In the View By: list, select Subgroup.
  4. Select a customer name from the Fleet drop-down list.
  5. Select a Site, folder , train or machine from the Hierarchy drop-down list.

Selecting the Filter Parameters

Each time you select a filter parameter, the bar or line chart at the top of the page and the Alarm List at the bottom of the page are updated. After you select the first filter parameter, the Show Filters icon displays a blue background, and the Clear Filters icon is displayed

To select the filter parameters, complete the following steps;

  1. In the Alarm table, click the Show Filters icon. The column headings in the Alarm table are modified. For example, the Last Occurred column displays a text box where you can enter a date. The Fleet Level column contains a drop-down menu where you can filter alarms by selecting  High, Low, Medium, or All (default).
  2. From the Data Source column drop-down, select System 1.
  3. From the Alarm List Range, drop-down, click 1 Month.
  4. From the Fleet Level column, click Medium.
  5. Option: Click the Clear Filters icon to restore the Alarms List default column settings.
