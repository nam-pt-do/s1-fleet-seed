Changing Performance Thresholds per Machine

The thermal performance index threshold for each machine can be configured based on the performance of the machine.

To change the thermal performance for a machine with node ID (22978) and enterprise ID ‘Platform_ea0c98ea-4805-4113-bad7-409810c77b07’, execute the following SQL commands in the pgadmin console:

  1. update fms.health_index set
  2. tp_med = 85.0,
  3. tp_high = 91.0

where path = 'Platform_ea0c98ea-4805-4113-bad7-409810c77b07/100/238/8267/22911/22919/22974/22978/';

 

NOTE : The path of the node must be the fully-qualified path of the node, including the node ID’s for all intermediate nodes. To query The mode path for a given node ID, issue the following SQL select path from fms.node,where enterprise_id = 'PlatformNERv15_ea0c98ea-4805-4113-bad7-409810c77b07' and id = '22978';
