Editing a User

To edit the properties of a current's user account settings, follow these steps:

  1. From the home page, click Administration. The Users page is displayed.
  2. Select the row that contains the name of the user you want to edit.
  3. Click Edit. The Edit User page is displayed.



  4. Complete one or more of the steps described in the table below to edit the user account settings.
  5. Click Save to apply the edits, or click Cancel to discard the edits.
    Changes made to the User Account settings take effect immediately.

Depending on System 1 Fleet Management configuration, you might see a Change Password link. Changing a user password from the Administration module only applies for local authentication installations of the System 1 Fleet Management software. For more information about local authentication, refer to the System 1 Fleet Management Installation Guide. For LDAP or Active Directory (AD) installations, user passwords are managed through the LDAP or Active Directory (AD) administrator.

To do this Follow these steps
Edit the Username Enter a new Username to be entered at login.
Edit the Name Enter a new Name to store in the database.

Edit the Role

 From the Role drop-down menu, select a new role, The available roles are User, Administrator, or SuperAdministrator.
Edit the Email address In the Email box, change the user's email address.
Edit Customer Access settings Select or clear one or more check boxes to change the customer(s) that the user can access.
Option: click Change Password Enter a password in the New Password field. Re-enter the New password
