Exporting Alarm Table Data

You can export the data that is in the Alarm Table and save the data in to a .csv file.

  1. From the Dashboard, click Alarms. The Alarms page is displayed.
  2. Click the down arrow in the Export icon.
  3. Select Export consolidated table or click Export unconsolidated table.

    If you select the Export consolidated table option, the Alarms List table is exported as shown. Alarm occurrences are consolidated into a single table row (see figure below).


    If you select the Export unconsolidated table option, you will export all alarm occurrences as separate table rows. This option could generate a large amount of data (see figure below).



  4. The exported table is saved to the Downloads folder. The default filename of the exported table is S1Fleet-Customer(xx)-enterprise-events (x).csv.
  5. To view the exported file, navigate to the Downloads folder. Open the .csv file in a spreadsheet application.
