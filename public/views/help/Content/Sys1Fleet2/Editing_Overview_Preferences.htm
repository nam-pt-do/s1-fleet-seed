Editing Overview Preferences

In this section of the Preferences page, you can change settings that affect the Overview page. The following illustration shows an example of the Overview section on the Preferences page.

Default View After Logging in to Fleet UI

Click Gauge View or Grid View.

Gauge View

The settings in this section control the appearance and organization of the gauges displayed after you click the gauge icon on the Overview page.

Gauge Options

System 1 Asset

Click one or more of the parameters in the following list. The permitted choices or combinations are:

 

  • Protection
  • Management
  • Protection and Management
  • Combined

System 1 Instrumentation

Click one or more of the following parameters. The permitted choices and combinations are;

  • Protection
  • Management
  • Protection and Management
  • Combined

SmartSignal

Click or clear SmartSignal.

Thermal Performance

Click or clear Performance.

Default Sort

Click one of the following sort parameters to set the default sort of information presented in the gauges. Although other parameters appear in the list, only the ones listed below can be selected for Default Sort;

  • Name
  • Asset Protection
  • Asset Management
  • Asset Combined
  • Instrumentation Protection
  • Instrumentation Management
  • Instrument Combined
  • SmartSignal
  • Performance

Grid View

The settings in this section control the appearance and organization of the information displayed after you click the grid icon on the Overview page.

Default Sort

Click one of the following parameters to set the default sort order of the data presented in the grid view.

  • Name
  • Source System
  • Severity Level
  • Total Alarms
