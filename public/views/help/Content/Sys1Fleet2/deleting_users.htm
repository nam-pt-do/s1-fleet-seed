Deleting a User

To delete a user from the system, follow these steps:

  1. From the home page, click Administration. The Users page is displayed.
  2. Locate the row that contains the name of the user you want to delete.
  3. From the Actions column, select edit. The Edit Users page is displayed.
  4. Click Delete User
  5. Click OK.

The Users page is displayed. The name of the deleted user is no longer listed on the Users page.
