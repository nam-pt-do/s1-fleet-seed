Administering Enterprises

On the Enterprise page, you can view information, edit enterprises, and search for an Enterprise.

Viewing Enterprise Information

You can view the following information on the Enterprises page, as shown in the following figure:

  • Enterprise Name - the name assigned to the Enterprise.
  • Source - the type of Enterprise, System 1 or SmartSignal.
  • Customer/Tenant ID - the Customer or Tenant ID name assigned to the associated enterprise
  • Status - Active or Inactive. Inactive Enterprises will not be displayed in the Dashboard pages.
  • Last Event Sync Time - Information (events) are continually being pushed to the Portal, and the information displayed is only as current as the last time the events in the source system and the portal was synchronized. This column contains the time of the latest sync occurrence.
  • Last Config Sync Time - The last time the configuration was synchronized between the source system and Fleet portal.

Editing an Enterprise

To edit Enterprise details, follow these steps;

  1. From the Dashboard, click Administration. The Users page is displayed.
  2. Click Enterprises. The Enterprises page is displayed.
  3. Select a row in the Enterprise table.
  4. Click Edit. The Edit Enterprise page is displayed. The Edit button is dimmed until the events and the configuration are totally synched between the source system and Fleet portal.
  5. Complete one or more of the edits described in the following table.
  6. Click OK to save the edits and close the Edit Enterprise page.
To do this Follow these steps
Change the Enterprise Name In the Display Name box, enter a new name. Click OK to save the changes. Click Cancel to restore the original settings. The new Enterprise Name is displayed in the Hierarchy pane on the Dashboard pane.
Change a Customer In the Customer drop-down menu, select a Customer name. Click OK to save the changes. Click Cancel to restore the original settings.

Change the Status of the Enterprise

 From the Status drop-down menu, select Active or Inactive. Click OK to save the changes. Click Cancel to restore the original settings. After you change the status to active, the name of the Enterprise is displayed in the Hierarchy drop-down list.

Searching for and Navigating an Enterprise

To search for Enterprises by name or other details, or to select a page from the list of enterprises, complete the following list:

  • Enter text in the Search box and click the magnifying glass icon.
  • Use the pagination box to navigate the list of Enterprises.
