Administering Users

Administration Module

The System 1 Fleet Management administrator is responsible for managing users, activating and deactivating enterprises, and mapping System 1 Fleet and SmartSignal Assets. Click the Administration module on the Dashboard. The module opens to the Users tab (see following figure).

  • From the Users page, administrators can add a user, delete a user, and edit parameters in the user profile.
  • From the Enterprises page, administrators can activate or deactivate an enterprise.
  • From the Mapping page, administrators can export SmartSignal and System 1 data to a .csv file, map a System 1 and SmartSignal Assets, and import the updated spreadsheet.
