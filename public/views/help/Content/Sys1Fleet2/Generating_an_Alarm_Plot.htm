Printing or Saving an Alarm Plot

Within an alarm plot pane, click the Chart Context Menu icon . The icon displays two selections: Print or Save as PNG.

Printing an Alarm Plot

  1. Click Print to print the selected alarm plot.
  2. If you select Print, the browser displays a print window. Make printing selections as required for your environment.
  3. Click Print.

Saving an Alarm Plot

  1. Click Save as .PNG. The alarm plot is saved to the Downloads directory. The default name of the downloaded alarm plot is download(x).
  2. To view the downloaded alarm plot, double-click the file name. The computer operating system displays a screen of viewer types that can display the alarm plot.
  3. Select a viewer type.

Downloading and Viewing a tag Grid

  1. Click the tag grid download icon The grid information is saved to the Downloads directory with the default file name of download(x).csv
  2. Navigate to the Downloads directory.
  3. Double-click the file name. The tag grid data opens in a spreadsheet (see the following figure).
