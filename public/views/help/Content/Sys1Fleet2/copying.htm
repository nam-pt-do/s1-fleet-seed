Printing or Saving a Chart or Plot

You have the option to save or print an alarm bar or line chart or a plot.

To print a chart or plot, complete the following steps;

  1. Generate the chart or plot that you want to print.
  2. Click the Chart Context menu icon . The icon is located in the right corner of the top line of the chart.
  3. Select Print. A Print Preview page is displayed.
  4. Enter the print parameters required for your location.
  5. Click the Print button on the Print Preview page to print the file.

Saving a bar or Line Chart or Alarm Plot as a .PNG File

The steps in this section explain how to save a bar or line chart or an alarm plot as a .png file directly from the Alarms page.

  1. Generate the chart or plot that you want to save.
  2. Click the Chart Context menu icon . The icon is located in the right corner of the top line of the chart or plot.
  3. Select Save as PNG. By default, the file is saved in the Downloads folder on your computer. The default name of the saved file is download. The file type is File. Change the extension of the file to .PNG and double-click the file to open it. .
  4. To view a saved chart or plot, go to the Downloads folder.
  5. Double-click a download file. The Open With menu is displayed (see the following figure).
  6. Select an application to display the chart or plot that you downloaded.
