Navigating Between the Overview and Alarm Pages

This topic explains how to use the drill-down features on the Grid View and the Gauges view pages to quickly access information on the Alarms page.

Using the Drill Down Feature From the Grid View

The information in this section explains how to use the drill-down links that are in the Grid View table (see the following figure).

The following table explains the call-outs in the figure above.

Figure Reference Number Description
1 Total Alarms: Click this link to open the Alarms page. You can generate a bar or line chart pertaining to all the alarms in all the folders displayed on the Overview page.
1

High, Medium or Low. Select a row in the Overview index table and click a number in the High, Medium or Low column. The action opens the Alarms page, where you can view a bar graph of the selected alarms. The default bar graph format is subgroup.
