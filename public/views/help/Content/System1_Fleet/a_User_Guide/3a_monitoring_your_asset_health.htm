Monitoring Your Asset Health

When you log into the application, the Grid View opens by default in the Overview page.

  1. Review the table to monitor the following:

    • Alarm severity level for your selected asset. The icon color represents the highest severity alarm.
    • Asset communication or running status. Enterprises and folders show communication status, while trains and machines show running status.
    • Active alarm counts by severity.

  2. Click a Total Alarms hyperlink to navigate to the corresponding Alarms page.

  3. To refresh the alarms list, click the or icon. If the icon contains a timer, the Auto-Refresh feature is active, but you can manually refresh the display at any time.
