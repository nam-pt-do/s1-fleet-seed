Creating and Saving a New Analysis Plot

You can create analysis plots for one or more points or tags, and any variable associated with the point or tag.

  1. Select the Analysis module from the menu bar. The blank landing page appears.

  2. The default time span for the Analysis module is one hour (1h). You can use that time span or select a new one:

    • Click the calendar icon and select an established time span from the list.

      or

    • Click Custom to open the calendars, then select start and end dates and times from the Start Date and End Date calendars and click Apply.

  3. Click New Plot. A blank New Chart screen opens.

  4. Click the drop-down arrow for the asset hierarchy and navigate to an asset.

  5. In the Points/Tags section, select a point or tag, then select one or more variables from the list on the right.

  6. Repeat for each point or tag to be analyzed. As you add points/tags and variables, the number next to Current Selections at the bottom of the page increases.

  7. Clickthe Current Selections hyperlink to display a table containing each selected point/tag and associated variable.

    To remove a point or tag, click the Delete button on the right. Otherwise, continue to step 8.

  8. Click Plot new chart. Each selected variable appears as a separate line in the resulting plot. The illustrated variables are identified in a legend on the right.

    Repeat as desired to create different time increments and plot combinations.

You can also use the following options:

  1. To refresh your plot at any time, click the icon.
  2. To save your plot as a graphic file or a private or public preset, see Saving Your Plot or Preset.
  3. To clear all chart plots from the application memory, click Open Plot Session, then click the Clear all charts button.
