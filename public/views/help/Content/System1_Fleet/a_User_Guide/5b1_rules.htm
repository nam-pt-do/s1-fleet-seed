Rules

Within the SmartSignal system, standard and diagnostic rules are defined in conditional statements that identify limits and thresholds where an alarm should be triggered. System 1 Fleet Management recognizes these rules and displays corresponding identifiers when you create SmartSignal alarm plots.

When you create an alarm plot, circle and diamond icons appear above the data lines to provide a quick view of data points where rules prompted an alarm:

  • Each circle icon () appearing in the SmartSignal plot indicates that a standard rule prompted an alarm.
  • Each diamond icon () indicates that a diagnostic rule prompted an alarm.
