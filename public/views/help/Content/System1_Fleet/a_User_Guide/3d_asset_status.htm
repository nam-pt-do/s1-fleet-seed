Asset Status

Asset status identifies where the following occurs:

  • Points for an enterprise or folder are or are not communicating.
  • A train or machine is or is not running.

In the Grid view, an oblong () icon appears in the Status column. You can pause on the icon to open a tooltip describing the status, as shown below.

In the Gauge view, a text alert is displayed to identify the type of non-operational status.

If there is no message, then the machine is considered running and communicating normally. The speed at which equipment is considered "running" depends on the equipment configurations at the site.

The following table explains the meaning of each colored icon:

If you select this in the Asset Hierarchy This Grid icon Means this
Enterprise/folder

The agent, proxy, system, or data acquisition server (DAQ) is not communicating.
Train/machine

Asset is not running.

Enterprise/folder All asset communication points are functioning properly.
Train/machine Asset is running. The running speed is specified in the tooltip.
Enterprise/folder Communication state unknown.
Train/Machine Running state unknown.
