SmartSignal Alarm Plots

SmartSignal alarms are based on efficiency models relating to turbine conditions such as temperature, pressure, speed, and so forth. Alarms (advisories) are triggered from deviations in these conditions from specified generic or diagnostic rules, which contain threshold data for each tag specific to the asset. Each tag has its own rule set.

SmartSignal data for the selected alarm is displayed in one or more time series trend graphs for the contributing tags, as shown below.

SmartSignal plots show the actual data line in blue, and an estimated data line in green. Estimations are calculated from historical data that represents how the machine should be operating.

When you pause on a data point, the actual and estimated values are displayed in a tooltip, along with a residual calculation, which is the difference between the actual and estimated values at the data point. Residuals are illustrated by icons above and below the data lines in the plot.

The chart also displays circle and diamond icons to illustrate the data points where a standard or diagnostic rules were triggered. For information, see Rules.
