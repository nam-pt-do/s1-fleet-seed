Saving a Plot or Preset

You can save any chart you create as a private or public preset to facilitate the creation of future charts. Presets are automatically saved to the server. You can also save your chart as a graphic or PDF file to your hard drive. From your local directory, you can print the file for use in a presentation.

  1. To save your plot as a preset, click Save As.

  2. In the Save new dialog box, enter a name for your plot preset. Use alphanumeric characters only.

  3. Select one of the following:

    • Private, to restrict the preset to your own use.
    • Public, to make the preset available for all users.
  4. To refresh your chart at any time, click the icon.

  5. To save your chart as a compatible file type, click the down arrow as shown below, then follow the prompts to create and save the new file.

  6. To clear all chart plots from the application memory, click Open Plot Session, then click the Clear all charts button.
