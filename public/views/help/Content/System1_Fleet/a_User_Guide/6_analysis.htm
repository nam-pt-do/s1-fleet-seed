Operational Analysis

The Analysis module allows you to select a System 1 point or SmartSignal tag, and run charts for different variables to illustrate machine operational data over different intervals.

For parameters that you examine frequently, you can save any custom plots you create as presets for future use. You can restrict your presets to your own view, or you can make them public.

You can also save your plots in a graphics format to include in presentations or discussions.
