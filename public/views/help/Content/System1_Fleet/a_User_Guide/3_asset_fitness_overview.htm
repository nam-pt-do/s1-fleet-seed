Asset Fitness Overview

When you log into System 1 Fleet Management, the Dashboard opens to the Overview tab. From Overview, you can navigate to an asset and access the Grid View, which provides a snapshot of asset fitness (the icon color reflects the highest active alarm level), along with alarm count breakdowns and the running or communication status.

  NOTE: An asset can be an enterprise, folder, train, or machine. For information, see Asset Hierarchy.

Identifiers are color-coded to indicate magnitude:

  • Alarm severity is identified by a square icon (). The icon color corresponds to the highest level alarm triggered on the selected asset.
  • Running and communication statuses are identified by an oblong icon ().

You can also navigate to additional alarm information and plots for trend and waveform data, and create charts to analyze data for selected points or tags over a designated interval. For information, see Individual Alarms, Alarm Plots, and Operational Analysis.
