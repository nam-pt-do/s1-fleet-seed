Application Administration > Users

Managing Users

As an application administrator, you can create, modify, and delete user accounts.

To add a user:

  1. From the dashboard, click the Administration tab. The Users page appears by default.

  2. Click Add New User to open the following screen. All fields are required.

  3. Enter the following information in each field:

    In this field Enter or select this

    Username

    		 The user's login identifier. Use between 8 and 20 characters e.
    Role

    Select one of the following:

    User: Can access the Dashboard features only.

    Administrator: Can manage users and enterprises. An administrator can add other administrators, but only the installation superuser can delete them.

     

    NOTE: A superuser role is assigned during the initial
    installation. The superuser can see and access all
    customers and tenants in a multi-tenant configuration. The superuser access the Administration module
    and perform all tasks associated with the
    administrator role.

    Name

    The user's display name.

    The user can modify the display name in the account profile.
    Email

    The user's email address.

    The user can modify the email address in the account profile.
    Password

    Enter a temporary password for the user. The user can change the password in the account profile, and should be advised to do so upon initial login.
    Customer access <placeholder> applicable to GE/Bently users only?

  1. Click Add User.

To edit a user:

  1. Click the Administration tab. The Users page opens by default.
  2. Select a user from the list and click the Edit button on the far right.
  3. Modify the fields as necessary, then click Save.

To delete a user:

  1. Click the Administration tab. The Users page opens by default.
  2. Select a user from the list and click the Edit button on the far right.
  3. Click Delete User.

  4. When the confirmation screen appears, click OK to confirm the deletion.
