Creating SmartSignal Alarm Plots

To generate a SmartSignal alarm plot:

  1. From the dashboard, click the Alarms tab.
  2. Select a SmartSignal alarm from the list and click the Alarm Plot icon to generate a chart with the default interval.

  3. From the Alarm List Range, select an interval.

    A revised chart appears.

  4. To open a table containing a list of tags contributing to the trend lines, click Show Tag Grid.

  1. To download the table to an Excel spreadsheet, click the Export icon, as shown below:

    You can export the data in a consolidated or unconsolidated format. A consolidated display replicates the data in the table, while an unconsolidated display includes each alarm occurrence.

  2. To close the table, click Hide Tag Grid.
