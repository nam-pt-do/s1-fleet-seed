Running a Plot from a Preset

For frequently-evaluated data, you can use an existing preset to create an analysis plot.

  1. From the Analysis module, click Open plot session.

  2. From the drop-down list on the left, select Public or Private.

    A public preset is available to all users. Private presets are available only to the user who created them.

  3. Click Apply to display the plot.

  4. To view the point/tag and variable data over a different time span, click the calendar icon and select new start and end dates and times.

You can edit the settings and save the preset under a new name. See Modifying a Plot or Preset.
