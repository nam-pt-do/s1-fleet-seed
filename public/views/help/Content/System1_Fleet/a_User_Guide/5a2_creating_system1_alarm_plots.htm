Creating System 1 Alarm Plots

When you select an alarm and click the icon, default trend and waveform plots appear when data is available. Using the drop-down menus in each plot, you can select different parameters to view different data representations.

System 1 plots differ from SmartSignal plots. For a better understanding of the data that appears in each, see System 1 Alarm Plots and SmartSignal Alarm Plots.

To generate a System 1 alarm plot:

  1. From the dashboard, click the Alarms tab.
  2. Select a System 1 alarm from the list.
  3. The default plot configurations appear. If there is no data to plot, a NO DATA FOR THIS POINT or NO WAVEFORM DATA FOR THIS POINT message appears in the corresponding plot area.

  4. To change the plot parameters, select a parameterfrom the drop-down lists above each plot. For information on the drop-down list contents, see System 1 Alarm Plots.

  5. To see the values of your data points, pause on a location in the plot. The values are displayed in the upper right corner of the plot.

  6. To view another alarm plot, click another row in the alarms grid and repeat steps 1 through 5.
