Alarms

The Alarms page contains the Alarms table with all active alarms and advisories, and the default alarm plot.

The Alarms table contains alarms or advisories triggered at each point for System 1 or each tag for SmartSignal. Points and tags correspond to where sensors for the alarm systems are installed on a machine to detect and collect operational time series data. When conditions are outside System 1 setpoint boundaries for a point, or when they violate SmartSignal rules defined for a tag, alarms are triggered.

System 1 Fleet Management identifies alarm or advisory severity according to the scale for the native system, and then normalizes the severity to High, Medium, or Low to provide a common scale in the Alarms page.

Bar icons, as shown below, are color-coded for each severity (System 1, indicated by S) or priority (SmartSignal, indicated by P).

Alarm plots are useful when investigating machine issues at measurable points or tags, which can range from temperature spikes to vibrations that exceed established thresholds.

When you open the Alarms page and select a System 1 or SmartSignal alarm, the corresponding alarm plots appear. You can select different plot parameters from the drop-down lists to customize the plots and tailor the data to fit your diagnostic needs.

Refer to System 1 Alarm Plots and SmartSignal Alarm Plots for more specific information on each type.
